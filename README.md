Система керування тікетами

Система керування тікетами — це комплексний додаток для роботи з підтримкою: керування тікетами, клієнтами та користувачами. Вона надає платформу для створення, отримання та оновлення тікетів, а також керування інформацією про клієнтів і користувачів. Система побудована на базі Laravel і використовує різні технології, включаючи Eloquent-моделі, сервіси та ресурси.

Функціонал

Керування тікетами: легке створення, отримання та оновлення тікетів

Керування клієнтами: управління інформацією про клієнтів та їхні тікети

Керування користувачами: управління акаунтами та правами доступу

Пошук та фільтри: пошук тікетів за статусом, датою, клієнтом тощо

Статистика: перегляд даних про тікети — щоденно, щотижнево, щомісяця та загалом

Керування медіа: завантаження файлів, прикріплених до тікетів

Технології

Laravel Framework: основний фреймворк додатку

Eloquent Models: робота з базою даних та зв’язками між таблицями

Services: інкапсуляція бізнес-логіки та спрощений інтерфейс для взаємодії з додатком

Resources: трансформація даних у JSON-відповіді для API

Spatie Media Library: керування та завантаження медіафайлів

Carbon: робота з датами та часом

Illuminate: різні компоненти Laravel — валідація, аутентифікація, кешування

Встановлення

Щоб встановити систему керування тікетами:

Клонуйте репозиторій командою git clone

Виконайте composer install для встановлення залежностей

Запустіть php artisan migrate для створення таблиць у базі даних

Виконайте php artisan db:seed для наповнення бази початковими даними

Налаштуйте параметри додатку у файлі .env

Використання

Щоб користуватися системою:

Запустіть додаток командою php artisan serve

Відкрийте веббраузер та перейдіть за адресою http://localhost:8000

Увійдіть у додаток за обліковими даними, створеними під час встановлення

Створюйте, отримуйте та оновлюйте тікети, клієнтів та користувачів за потреби
